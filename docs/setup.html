Step 3: 自動通知を有効化
以下の自動通知トリガーを設定します。
  • ● 毎朝 8:55 に当日の予定を LINE に送信
  • ● 各予定の5分前にリマインダーを送信
  • ● トリガーは GAS のタイムベーストリガーとして登録されます
Step 4: LINE Webhook を設定
以下の Webhook URL を LINE Developers に登録してください。
https://script.google.com/macros/s/AKfycbwQ1Vz9di8MD4ADi1kcVfvUdXyta9uVRN_Vmf5sDDNU5GrVV6DmSMZbcAG9zV_-XPMG/exec
  1. LINE Developers のチャネル設定を開く
  2. 「Messaging API設定」タブを選択
  3. 「Webhook URL」に上記の URL を貼り付けて「更新」
  4. 「Webhookの利用」をONに切り替え
  5. 「検証」ボタンを押して「成功」と表示されることを確認
