FEATURES

CalPush でできること

日々のスケジュール管理をLINEで完結

📅

毎朝の予定通知

毎日 8:55 に当日のスケジュールを LINE に自動送信。朝の予定確認が一目で完了。

⏰

5分前リマインダー

会議やイベント開始の5分前に LINE でお知らせ。うっかり忘れを防止します。

📨

メール着信通知

指定したドメインやアドレスからのメールを検知し、LINE に即時通知。

💬

LINE からスケジュール操作

「今日の予定」「予定追加」などのチャットコマンドで手軽に操作。

💻

PC 管理ダッシュボード

ブラウザから週間スケジュール管理、メール通知設定、空き枠管理が可能。

📋

空き枠仮押さえ

候補日時を一括仮押さえ。確定時に他の候補を自動解放するグループ管理機能。
